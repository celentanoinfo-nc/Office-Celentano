Einmalige Einrichtung
Verbindet alle Geräte über Google Sheets.
Brauchen Sie nur einmal zu machen.
1Gehen Sie auf sheets.google.com → Neues leeres Sheet anlegen → beliebigen Namen geben (z.B. „Celentano Lager")
2Im Sheet oben: Erweiterungen → Apps Script
3Alles im Editor löschen → den Code aus der zweiten Datei (gas_script.js) hineinkopieren → Speichern (Disketten-Symbol)
4Oben rechts: Bereitstellen → Neue Bereitstellung → Typ: Web-App → „Ausführen als": Ich → „Zugriff": Jeder → Bereitstellen → Google-Konto bestätigen
5Die angezeigte Web-App-URL (beginnt mit https://script.google.com...) hier einfügen:
